Courses related to Information Technology Bachelors

ITM 301

Introduction to OS and Hardware, taught by Professor Pappademetriou

served as an introduction to hardware and software concerning computers, including: hardware and it's components, troubleshooting and system repair.

ITM 311

This class was named Intro to Software development, and taught by Professor Papademas. It served as a basic introduction to Java, a programming language. The class was set up to be taught more in depth at a later date.

PHYS 200

This class was formally called Energy, Waves, Materials and Forces. it was taught by Professor Laurent-Muehleisen and was based around learning physics as a architecture student. It was recommended to ITM students to fill a science requirement.

HUM 200

This class was one of many Humanities 200 options freshmen could take to fufill the basic English requirement. It could also be filled by prior high enough AP credit. This class was taught by Professor Fisher, and focused heavily on Civil disobediance of the past and future.

ITM 312

This class was called Intro Systems Software Programming and taught by Professor Hawkins. It is considered the C++ version of ITM 311 and taught you the basics of C++, This class was meant to be taught after 311, but some people took it before with enough credits.

EG 255

This class is formally called Engineering Graphics for Non-Engineers. This class is taught by Professor Novak and is meant to be taken as a science credit. It teaches drafting, including CAD and AutoCAD.

PSYC 301

This class is Industrial Psychology and is meant to teach workplace psychology and how people use it. This class is a requirement for everyone, as students will usually have atleast one desk job in their lifetime. The class was taught by Dr. Bauer, who studies this for a living, and included a lot of group work.

ITMO 456

This class was called Introduction to Open Source Operating Systems and taught Linux in a lab setting. Homework assignments included daily labs, and while there was no real homework that couldn't be taught in class students spend a majority of their time doing labs and listening to the teacher. Lab is open Monday-Friday until 5. The class was taught by Professor Hughes-Durkin.
